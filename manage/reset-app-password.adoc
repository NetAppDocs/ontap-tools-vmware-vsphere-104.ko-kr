= 애플리케이션 사용자 비밀번호 재설정
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
애플리케이션 사용자 비밀번호는 vCenter Server에 SRA 및 VASA 공급자를 등록하는 데 사용됩니다.

.단계
. 웹 브라우저에서 ONTAP 도구 관리자를 실행합니다. `\https://<ONTAPtoolsIP>:8443/virtualization/ui/`
. 배포 중에 제공한 ONTAP tools for VMware vSphere 로 로그인합니다.
. 사이드바에서 *설정*을 선택하세요.
. *VASA/SRA 자격 증명* 화면에서 *비밀번호 재설정*을 선택합니다.
. 새로운 비밀번호를 입력하고 새 비밀번호 입력을 확인하세요.
. 변경 사항을 적용하려면 *재설정*을 선택하세요.
