= 플레이스홀더 데이터 저장소 구성
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
보호된 가상 머신(VM)의 복구 사이트에서 vCenter 인벤토리에 자리가 있도록 플레이스홀더 데이터스토어를 구성해야 합니다. 플레이스홀더 VM은 크기가 작고 수백 KB 이하의 용량만 사용하므로 플레이스홀더 데이터스토어는 크게 구성할 필요가 없습니다.

.시작하기 전에
* 보호 사이트와 복구 사이트를 연결했어야 합니다.
* 리소스 매핑을 구성했어야 합니다.


.단계
. vCenter Server에 로그인하고 *사이트 복구* > *사이트*를 선택합니다.
. 보호된 사이트를 선택하고 *관리*를 선택하세요.
. 관리 탭에서 *플레이스홀더 데이터스토어* > *새로 만들기*를 선택하여 새로운 플레이스홀더 데이터스토어를 만듭니다.
. 적절한 데이터 저장소를 선택하고 *확인*을 선택합니다.
+

NOTE: 플레이스홀더 데이터 저장소는 로컬 또는 원격일 수 있으며 복제해서는 안 됩니다.

. 복구 사이트에 대한 플레이스홀더 데이터 저장소를 구성하려면 3~5단계를 반복합니다.
